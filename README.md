# Chat App
Real-time chat app, you can use by generating your unique id then you can contact with multiple persons at the same time or individual one. 

## Front-End Technologies: 
- Angular
- Socket-io-client
- Local Storage

## Backend Technologies:
- Node.js
- Socket-io
